gfx Library

!!ONLY USE THIS ON RENDER!!


gfx.color([number]r,[number]g,[number]b,[number,optional]a)
Sets the color to use (RGBA) 0 to 255

gfx.rect([number]positionX,[number]positionY,[number]sizeX,[number]sizeY)
Draws a rectangle

gfx.drawRect([number]positionX,[number]positionY,[number]sizeX,[number]sizeY, [number]lineWidth)
Draws the outline of rectangle

gfx.roundRect([number]positionX,[number]positionY,[number]sizeX,[number]sizeY, [number]radius, [number]quality)
Draws a rectangle with round corners

gfx.circle([number]positionX,[number]positionY,[number]radius, [number]quality])
Draws a Circle

gfx.text([number]positionX,[number]positionY,[string]text,[number,optional]scale)
Draws a string onto the screen

gfx.item([number]positionX,[number]positionY,[number]ItemLocation,[number,optional]scale)

Draws an Item from inventory, See Item for location, Do not guess location

gfx.image([number]positionX,[number]positionY,[number]sizeX,[number]sizeY, [string]filepath)
Draws an image from Scripts/Data

gfx.unloadimage([string]filepath)
Unloads an image from memory, only custom images

gfx.texture([number]positionX,[number]positionY,[number]sizeX,[number]sizeY, [string]mc_texture_path)
Draws a minecraft texture, same path as resource pack, textures/blocks/stone.png

gfx.effect([number]positionX,[number]positionY,[number]sizeX,[number]sizeY, [number]effectID)
Draws the icon of a potion effect

gfx.triangle([number]positionX1,[number]positionY1,[number]positionX2,[number]positionY2, [number]positionX3,[number]positionY3)
[3D-2D] Draws a triangle with the 3 points, go counter clockwise

gfx.quad([number]positionX1,[number]positionY1,[number]positionX2,[number]positionY2, [number]positionX3,[number]positionY3, [number]positionX4,[number]positionY4)
[3D-2D] Draws a quad with the 4 points, go counter clockwise
