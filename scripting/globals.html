Globals

getClipboard()
Returns the user's clipboard (string)

setClipboard([string]text)
Sets the user's clipboard


importLib([string]name)
runs a file from the Scripts/Libs folder


tableToJson([table]table)
Converts a table to a json string

jsonToTable([string]json)
converts a json string into a table


sendLocalData([string]identifier, [string]message)
sends a message to other modules/command
the identifier can be anything and you check if it match the one it should on the receiver side
only string can be sent this way, be careful for infinite loops



event.listen([string]event,[function]function)
Listens for events


events:
Event Description Function args.
KeyboardInput Listens for keyboard inputs key int, the pressed key
down bool, returns true if the key was pressed, false if not
MouseInput Listens for mouse inputs key int, the pressed key
down bool, returns true if the key was pressed, false if not
ChatMessageAdded Listens ingame chat message string, the content
username string, username of the sender
type number, message type
LocalDataReceived Receives Data sent by a command or a module identifier string, id of the data received
content string, data content
