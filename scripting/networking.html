Networking library

network.get(url, identifier)
Does a web request to the specified url

network.get(url, identifier, headers)
Does a web request to the specified url with headers

network.fileget(filepath, url, identifier)
Does a web request to the specified url and stores the result in the file, data will be filepath

network.fileget(filepath, url, identifier, headers)
Does a web request to the specified url with headers and stores the result in the file, data will be filepath



Identifier - second parameter of onNetworkData, headers - string[], list of headers.
